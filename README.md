# Anime List 🎥

Projeto React que lista animes usando a API AniList.

## 🚀 Tecnologias
- React + TypeScript
- Styled-components
- Jest + Cypress
- API GraphQL AniList

## 📦 Como rodar?
1. Clone o repositório: `git clone repo-url`
2. Instale as dependências: `npm install`
3. Rode a aplicação: `npm run dev`
